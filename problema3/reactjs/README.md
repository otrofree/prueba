La aplicación compilada

## src
El código fuente para Reactjs


**Para probar compilado** 
```
cd reactjs
yarn install
http-server buildApp
```

Después habir tu navegaor esta dirección [http://127.0.0.1:8080/](http://127.0.0.1:8080/)

**Para desarrollar** 
```
yarn start
```
